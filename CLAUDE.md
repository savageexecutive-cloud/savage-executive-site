# Savage Executive Site

Next.js 15 marketing site for savageexecutive.com. Rebuilt from Kajabi.

## Tech Stack
- Next.js 15 (App Router) + TypeScript
- Tailwind CSS 4
- Deployed on Vercel

## Brand
- Dark theme: `#0a0a0a` bg, `#c9a227` gold accents
- Fonts: Cormorant Garamond (display), Source Sans 3 (body)
- Voice: Confident, direct, faith-grounded, practitioner not theorist

## Key Pages
- `/` — Home (hero, story, services, stats, CTA)
- `/about` — Bio, turnaround story, credentials
- `/speaking` — Speaking topics, venues, differentiators
- `/podcast` — YouTube embed + platform links
- `/executive-partnership` — 3-tier pricing, all → Calendly
- `/ceo-cashflow` — Course landing + waitlist
- `/ministry` — Consulting inquiry form

## Integrations
- Calendly: Book a Clarity Call (all CTAs link externally)
- Stripe: Infrastructure ready, not live yet
- Forms: `/api/contact` and `/api/waitlist` (TODO: wire to email service)
